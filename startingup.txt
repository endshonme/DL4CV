jupyter notebook --no-browser --port=8889

Start another terminal for port forwarding:
forwarding 8089 port to local machine port: 8081

ssh -i privatekey -N -f -L localhost:8081:localhost:8889 jyotirmaysenapati@<ip>

open another terminal for genarl modification directly in the terminal of cloud.

ssh -i <privatekey_location> jyotirmaysenapati@<ip> 


send file from local machine to google cloud or any remote machine:

scp -i privatekey <space separated file names>  jyotirmaysenapati@<IP>:dl4cv_proj/DL4CV/
